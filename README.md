# mouseServo #
RPi + usbCam + opencv + mouse + servoBlaster 

## Demo
https://www.youtube.com/watch?v=hkiwkMcE_tI


## Requirements
1. opencv 2.4.x
2. servoBlaster
3. mouse
4. two servos
5. UVC compatible usb camera 


## Build Instruction:  ## 
g++ -o ./mouseServo0 ./mouseServo0.cpp `pkg-config --cflags --libs opencv`
